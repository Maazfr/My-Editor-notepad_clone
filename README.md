**Simple Notepad Clone – Java Text Editor**

This project is a basic text editor application developed using **Java AWT and Swing**, inspired by the functionality of the classic Windows Notepad. The goal of the project was to understand how desktop applications work in Java and to practice building graphical user interfaces using event-driven programming.

The editor provides a minimal and clean interface that allows users to **create, edit, and view plain text files**. Core features include opening existing text files, editing content within a text area, and saving files to the local system. The application also demonstrates the use of common GUI components such as **JFrame, JTextArea, JMenuBar, and JFileChooser**, which are commonly used when developing desktop applications in Java.

Through this project, key concepts of **Java GUI development, file handling, and user interaction** were explored. The implementation focuses on maintaining a simple and lightweight design while replicating the essential functionality of a standard text editor.

This project serves as a practical example of how Java can be used to build desktop utilities and highlights the use of Swing components to create interactive applications.
